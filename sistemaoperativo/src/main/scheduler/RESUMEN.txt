  ✓ SJFScheduler.java     - Shortest Job First
  ✓ RoundRobinScheduler   - Con quantum configurable

ESTRUCTURA

  ✓ SimulationClock      - Reloj sincronizado thread-safe
  ✓ ReadyQueue           - Cola con locks (ReentrantLock)
  ✓ SchedulerFactory     - Factory pattern

  MÉTRICAS CALCULADAS
═══════════════════════════════════════════════════════════════

Para cada proceso se calcula:
  • Tiempo de Espera     = Tiempo Retorno - Tiempo CPU
  • Tiempo de Retorno    = Tiempo Final - Tiempo Llegada
  • Tiempo de Respuesta  = Primera Ejecución - Tiempo Llegada
